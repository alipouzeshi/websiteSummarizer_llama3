🌐 Website Summarizer
An AI-powered Jupyter Notebook that extracts and summarizes website content using Ollama and Llama 3 – all running locally with no API costs.

✨ Features
🕷️ Web Scraping – Fetches and cleans text from any URL using BeautifulSoup

🤖 Local LLM – Uses Ollama with Llama 3 for private, offline summarization

📝 Smart Summaries – Generates concise, readable summaries focusing on key topics and purpose

🎨 Markdown Output – Displays formatted summaries directly in Jupyter notebooks

⚡ Lightweight – Runs locally without cloud dependencies or API keys
